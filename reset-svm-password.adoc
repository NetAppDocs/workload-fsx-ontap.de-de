= Setzen Sie das Speicher-VM-Passwort in NetApp Workload Factory zurück
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Setzen Sie bei Bedarf das Kennwort für eine Speicher-VM in NetApp Workload Factory zurück.

.Schritte
. Melden Sie sich mit einem der link:https://docs.netapp.com/us-en/workload-setup-admin/console-experiences.html["Konsolenerfahrungen"^]an.
. Wählen Sie das Menü aus image:workloads-menu-icon.png["Über das Hamburger-Menüsymbol kann man zu Workloads wie Speicher, EDA, KI, Datenbanken, VMware und Administration navigieren."] und wählen Sie dann *Speicher* aus.
. Wählen Sie im Speichermenü *FSx für ONTAP* aus.
. Wählen Sie in *FSx for ONTAP* das Aktionsmenü des Dateisystems mit dem Speicher-VM-Passwort aus, das zurückgesetzt werden soll, und wählen Sie dann *Verwalten*.
. Wählen Sie in der Dateisystemübersicht unter „Speicher-VMs“ die Option „Verwalten“.
. Wählen Sie im Bildschirm „Speicher-VMs verwalten“ das Aktionsmenü der Speicher-VM aus, deren Kennwort Sie zurücksetzen möchten, und wählen Sie dann *Grundlegende Aktionen* > *Kennwort zurücksetzen*.
. Geben Sie im Dialogfeld Kennwort zurücksetzen Folgendes ein:
+
.. *Neues Passwort*: Geben Sie ein neues Passwort für die Speicher-VM ein.
.. *Passwort bestätigen*: Geben Sie das neue Passwort zur Bestätigung erneut ein.


. Wählen Sie *Anwenden*.
